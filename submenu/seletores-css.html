Seletores CSS

Conheça os principais tipos de seletores utilizados em CSS.

1. Seletor de Elemento

Seleciona todos os elementos de um tipo específico.

p { color: blue; }

2. Seletor de Classe

Seleciona todos os elementos com a classe especificada.

.destaque { font-weight: bold; }

3. Seletor de ID

Seleciona um único elemento com o ID especificado.

#cabecalho { background-color: gray; }

4. Seletor Universal

Seleciona todos os elementos da página.

* { margin: 0; padding: 0; }

5. Seletor de Descendente

Seleciona elementos dentro de outros elementos.

div p { color: green; }

6. Seletor de Filho Direto

Seleciona apenas os elementos filhos diretos de um elemento pai.

ul > li { list-style: none; }

7. Seletor de Atributo

Seleciona elementos que possuem um atributo específico.

input[type="text"] { border: 1px solid #ccc; }

8. Seletor de Irmão Adjacente

Seleciona um elemento imediatamente após outro.

h1 + p { font-style: italic; }
