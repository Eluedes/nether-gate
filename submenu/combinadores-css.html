Combinadores CSS

Conheça os principais combinadores CSS e como eles permitem selecionar elementos de forma avançada usando relações entre tags.

1. Agrupamento de Seletores

O agrupamento de seletores permite aplicar o mesmo conjunto de regras CSS a vários elementos diferentes, separando os seletores por vírgula. Isso evita repetição de código e facilita a manutenção do estilo.

Exemplo:

h1, p, div {
  color: blue;
}

2. Seletores de Classe em Elementos Específicos

Você pode aplicar estilos apenas a elementos de uma determinada tag que possuam uma classe específica, utilizando o seletor de classe junto ao nome da tag.

Exemplo:

a.texto {
  text-decoration: underline;
}
