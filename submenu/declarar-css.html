Formas de Declarar CSS

Aprenda os três métodos principais para aplicar estilos em páginas HTML.

1. CSS Inline

Estilos aplicados diretamente no elemento HTML usando o atributo style

Exemplo:

<p style="color: red;">Texto em vermelho</p>

2. CSS Interno

Estilos escritos dentro da tag <style> no cabeçalho do documento.

Exemplo:

<style>
  p { color: blue; }
</style>

3. CSS Externo

Estilos definidos em um arquivo separado e vinculados com a tag <link>

Exemplo:

<link rel="stylesheet" href="estilos.css">
